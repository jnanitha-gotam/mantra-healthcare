# 🏥 NovaCare Medical Centre Website

A modern, responsive healthcare website developed using HTML, CSS, and JavaScript. The website provides essential information about the medical centre, its services, doctors, and contact details while allowing patients to book appointments through a user-friendly interface.

---

## 📌 Project Overview

NovaCare Medical Centre is designed to provide visitors with a clean, professional, and responsive healthcare website experience. It offers information about medical services, experienced doctors, and easy appointment booking.

---

## ✨ Features

- Responsive design for desktop, tablet, and mobile
- Modern navigation bar with hamburger menu
- Attractive hero section
- About NovaCare Medical Centre
- Medical services section
- Experienced doctors page
- Appointment booking form
- Contact page with Google Maps
- Tawk.to live chat integration
- SEO-friendly HTML structure
- Font Awesome icons
- Fast loading and clean UI

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts (Poppins)
- Google Maps Embed
- Tawk.to Live Chat

## 🚀 How to Run the Project

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Install the Live Server extension (optional).
4. Open `index.html`.
5. Right-click and select **Open with Live Server**.
